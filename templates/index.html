Features Used

The model analyzes 30 numerical features computed from digitized images of breast mass:

  • Mean values (10): radius, texture, perimeter, area, smoothness, compactness, concavity, concave points, symmetry, fractal dimension
  • Standard error (10): Standard error of the above measurements
  • Worst values (10): Worst/largest values of the above measurements
